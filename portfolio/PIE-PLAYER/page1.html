🥧PIE Player

Programmable Integrated Exhibition Player — lightweight web-native digital signage with a zero-backend philosophy.
Project Website

📖 Description

The PIE Player (Programmable Integrated Exhibition Player) is an out-of-the-box advertisement loop system built with modern web technologies. It separates management from presentation so a secondary control device can update a live display in real time.

🚀 Key Features

Dual-Page Architecture

  • The Kitchen (Manager): UI to structure playlists, add media URLs, and set timings.
  • The Plate (Display): Full-screen, distraction-free playback.

Intelligent Media Handling

  • Images: Customizable display durations.
  • Videos: Auto transition on end-of-file via onended.

Transitions & Sync

  • Hardware-accelerated fade animations for smooth transitions.
  • Instant-Sync using the browser StorageEvent API.

Pure Vanilla Stack

  • Built with HTML5, CSS3, and plain JavaScript. No libraries or backend required.

🛠️ How It Works

  1. The app runs fully in the browser using localStorage to persist playlists.
  2. Once a playlist is saved ("Baked"), the Display page loops through items continuously.
  3. Images use setTimeout; videos use onended for precise handoff.

📂 File Structure

  • manage.html — Configuration dashboard to build playlists.
  • display.html — Output screen for playback.

📥 Installation & Setup

  1. Copy manage.html and display.html into the same folder.
  2. Open display.html (press F11 for full-screen).
  3. Open manage.html on a secondary device or tab, add media URLs, and click Apply & Save.
